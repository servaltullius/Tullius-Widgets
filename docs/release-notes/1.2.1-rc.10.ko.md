# Tullius Widgets v1.2.1-rc.10

## 변경 요약
- 한국어 Skyrim 날짜/시간 위젯 표기를 로어 월 이름 음역 대신 숫자 월 형식으로 정리했습니다. 이제 게임 날짜가 `4E 201년 8월 21일 14:35`처럼 표시됩니다.
- 시간 위젯 표시 폭을 넓혀 긴 날짜 문자열이 좌상단 HUD에서 `...`로 잘리지 않도록 보정했습니다.
- 한국어 숫자 월 포맷과 시간 위젯 폭 동작을 검증하는 프런트 테스트를 추가했습니다.

## 사용자 영향/호환성
- 한국어 HUD를 사용하는 경우에만 날짜 표기가 더 자연스러운 숫자 월 형식으로 바뀝니다.
- 영어 및 커스텀 언어팩은 기존 `{monthName}` 기반 포맷을 계속 사용할 수 있습니다.
- UI 자산 변경만 포함되어 네이티브 DLL 동작 호환성에는 영향이 없습니다.

## 설치/업데이트 안내
1. `v1.2.1-rc.10` 프리릴리즈 zip을 다운로드합니다.
2. 압축 안의 `SKSE/`와 `PrismaUI/` 폴더를 게임의 `Data/` 폴더에 덮어씁니다.
3. 게임을 완전히 종료한 뒤 다시 실행해 갱신된 UI 자산이 로드되도록 합니다.
