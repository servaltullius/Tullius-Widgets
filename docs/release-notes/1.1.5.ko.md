# Tullius Widgets v1.1.5

## 변경 요약
- 플레이어 정보 위젯의 체력/매지카/스태미나 표기를 보강했습니다.
  - 일부 전투 상황(큰 피해/연속 피해 등)에서 엔진 값이 순간적으로 0 미만으로 계산되어 음수로 표시될 수 있던 케이스를 방지하기 위해, 표시값을 0 이상으로 클램프합니다.

## 사용자 영향/호환성
- 기존 설정 파일(`Data/SKSE/Plugins/TulliusWidgets.json`) 및 프리셋(`TulliusWidgets_preset.json`)과 하위 호환됩니다.
- Prisma UI 브릿지(`updateStats`, `updateSettings`) 스키마 변경은 없습니다.
- 참고: 체력/매지카/스태미나 위젯은 퍼센트(%)가 아니라 **현재 수치(포인트)** 를 표시합니다.
  - 퍼센트 값은 시각 알림(임계값 경고) 판단에만 사용됩니다.

## 설치/업데이트 안내
1. `TulliusWidgets-v1.1.5.zip`을 다운로드합니다.
2. 압축을 해제한 뒤 `Skyrim Special Edition/Data/` 경로에 파일을 덮어씁니다.
3. 업데이트 후 게임을 완전히 재시작해 DLL/UI가 함께 갱신되도록 해주세요.
