# Tullius Widgets v1.1.7-rc.2

## 변경 요약
- 경험치/레벨업 반영 타이밍과 스탯 동기화 파이프라인을 리팩터링했습니다.
  - 지연 반영 스케줄러를 단일 due-time 소비 방식으로 교체해, 레벨업 직후 XP 바가 늦게 갱신되던 케이스를 완화했습니다.
  - 스탯 전송을 `fast/full` 이중 주기로 분리해 고빈도 구간의 부담을 줄였습니다.
- 설정 저장 경로를 비동기 coalescing 저장으로 변경했습니다.
  - UI에서 연속 설정 변경 시 디스크 I/O 병목을 줄이고 프레임 저하 가능성을 낮췄습니다.
- 브릿지와 UI 업데이트 경로를 정리했습니다.
  - `window.TulliusWidgetsBridge.v1.*` 네임스페이스를 추가하고, 기존 전역 함수 키와 동시 호환되도록 유지했습니다.
  - 설정 path 업데이트와 표시 변경 감지 시그니처를 경량화해 불필요한 전체 직렬화 비용을 줄였습니다.

## 사용자 영향/호환성
- 기존 설정 파일(`Data/SKSE/Plugins/TulliusWidgets.json`) 및 프리셋 파일(`TulliusWidgets_preset.json`)과 하위 호환됩니다.
- 기존 브릿지 키(`updateStats`, `updateSettings`, `updateRuntimeStatus`)는 그대로 동작합니다.
- fast 동기화 payload에서 `timedEffects`가 생략될 수 있으며, UI는 이전 효과 목록을 유지하도록 보완되었습니다.
- 본 버전은 프리릴리즈(`rc`)이며, 실사용 전 기존 Data 폴더 백업을 권장합니다.

## 설치/업데이트 안내
1. `TulliusWidgets-v1.1.7-rc.2-ci.zip` 파일을 다운로드합니다.
2. 압축을 해제한 뒤 `Skyrim Special Edition/Data/` 경로에 덮어씁니다.
3. 게임을 완전히 종료 후 재실행해 DLL/UI가 함께 갱신되도록 적용합니다.
