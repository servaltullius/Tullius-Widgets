# Tullius Widgets v1.1.8-rc.8

## 변경 요약
- 코드 품질 및 안정성 전반 개선 (심층 분석 기반)
- C++ 네이티브:
  - `CollectStats()` 전체를 try-catch로 래핑하여 게임 API 비정상 반환 시 CTD 방지
  - 크리티컬 확률 계산에서 NaN/Infinity 방어 체크 추가
  - 타이밍 이펙트 수집 시 UI 싱글톤 미초기화 상태 안전 처리
  - 프리셋 임포트 실패 시 원인 로깅 추가
  - JSON 이스케이프 메모리 예약 크기 개선
- 프론트엔드:
  - `StatWidget`에 `React.memo` 적용으로 불필요한 리렌더링 대폭 감소
  - 위젯 드래그에 `requestAnimationFrame` 적용으로 반응성 개선
  - 위젯 위치 콜백 의존성 최적화 (드래그 중 콜백 재생성 방지)
  - JSON 파싱 실패 시 위젯 상태를 안전하게 초기화
  - 브릿지 핸들러 등록 시 예외 처리 추가
  - `normalizeGameTime` 이름 충돌 해소

## 사용자 영향/호환성
- C++ 네이티브 + 프론트엔드 양쪽 변경이므로 DLL과 프론트엔드를 함께 업데이트해야 합니다.
- 기존 설정 파일과 완전 호환됩니다.
- 위젯 레이아웃이나 기능 변경은 없습니다.

## 설치/업데이트 안내
1. 프리릴리즈 페이지에서 `v1.1.8-rc.8` zip을 다운로드합니다.
2. `Data/` 폴더에 덮어쓴 뒤 게임을 완전히 종료/재실행합니다.
