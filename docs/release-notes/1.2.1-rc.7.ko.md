# Tullius Widgets v1.2.1-rc.7

## 변경 요약
- 설정 패널 hotkey 경로를 게임 task 기반으로 다시 정리해, 패널을 연 직후 PrismaUI 포커스를 다음 틱에서 다시 잡도록 보강했습니다.
- 이 변경으로 패널은 열리지만 마우스 커서가 나타나지 않아 설정 UI를 조작할 수 없던 문제를 추가로 완화했습니다.
- `v1.2.1-rc.6`에서 수정한 포토모드/free camera HUD 숨김 보강은 그대로 유지됩니다.

## 사용자 영향/호환성
- 이번 RC는 `v1.2.1-rc.6` 이후 남아 있던 설정 패널 입력 회귀를 바로잡는 핫픽스입니다.
- 설정 패널을 열 때 입력 포커스 전환이 한 틱 늦게 적용되도록 바뀌었지만, 실제 체감상으로는 더 안정적으로 커서가 나타나는 방향을 의도한 변경입니다.
- 배포 구조와 설정 파일 형식은 유지됩니다. 네이티브 DLL과 Prisma UI 자산은 함께 업데이트하는 것을 권장합니다.

## 설치/업데이트 안내
1. `v1.2.1-rc.7` 프리릴리즈 zip을 다운로드합니다.
2. 압축 안의 `SKSE/`와 `PrismaUI/` 폴더를 게임의 `Data/` 폴더에 복사하거나 모드 매니저로 그대로 설치합니다.
3. 게임을 완전히 종료한 뒤 다시 실행해 DLL과 UI 자산이 함께 새로 로드되도록 합니다.
