# Tullius Widgets v1.1.8-rc.5

## 변경 요약
- 무기 데미지 표시 정확도를 개선했습니다.
  - 기존 자체 계산(`baseDmg × skillMult × perkMult`)을 게임 엔진 API(`PlayerCharacter::GetDamage`)로 교체해 대장간 강화, 인챈트, 포션 효과가 모두 반영됩니다.
  - 인벤토리 화면에 표시되는 값과 동일한 결과를 보여줍니다.
- 크리티컬 확률이 비전투 시에도 퍼크 보너스를 반영하도록 수정했습니다.
  - 전투 타겟이 없을 때 플레이어 자신을 fallback target으로 사용해 `BGSEntryPoint` 평가를 항상 실행합니다.
- 모든 스탯 업데이트에 `timedEffects`를 포함하도록 변경했습니다.
  - 기존 kFast/kFull 이중 모드를 제거해 활성 효과 표시의 최대 300ms 지연을 해소했습니다.

## 사용자 영향/호환성
- 프론트엔드 변경 없음 — JS 측 데이터 구조는 동일합니다.
- 대장간 강화 무기 장착 시 위젯에 표시되는 데미지가 인벤토리 값과 일치하게 됩니다.
- 비전투 상태에서도 크리티컬 확률 퍼크(예: Assassin's Blade)가 즉시 반영됩니다.
- 기존 설정 파일과 완전 호환됩니다.

## 설치/업데이트 안내
1. 프리릴리즈 페이지에서 `v1.1.8-rc.5` zip을 다운로드합니다.
2. `Data/` 폴더에 덮어쓴 뒤 게임을 완전히 종료/재실행합니다.
3. 대장간 강화 무기를 장착해 인벤토리 표시값과 위젯 값이 일치하는지 확인합니다.
