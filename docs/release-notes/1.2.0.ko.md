# Tullius Widgets v1.2.0

## 변경 요약
- Photo Mode 진입/복귀 시 위젯 숨김 판정을 보강해 HUD 표시 안정성을 높였습니다.
- 레벨업 전후 XP 위젯에서 발생하던 텍스트 잘림과 글자 중복 표시를 수정했습니다.
- 네이티브 stats 수집과 프론트엔드 표시 경로를 함께 정리해 전반적인 런타임 안정성을 개선했습니다.
- 화면 효과, 시간 표시, 설정 제어 등 일부 UI 구성요소의 렌더 동작을 다듬었습니다.

## 사용자 영향/호환성
- 이번 버전은 C++ 네이티브 DLL과 Prisma UI 프론트엔드가 함께 변경되므로 두 결과물을 같이 업데이트해야 합니다.
- 기존 설정 파일 경로와 저장 형식은 유지됩니다.
- 위젯 배치 개념이나 기본 사용 방식은 바뀌지 않으며, 표시 안정성 개선이 중심입니다.

## 설치/업데이트 안내
1. `v1.2.0` 배포 zip을 다운로드합니다.
2. 압축 안의 `SKSE/`와 `PrismaUI/` 폴더를 게임의 `Data/` 폴더에 복사하거나 모드 매니저로 그대로 설치합니다.
3. 게임을 완전히 종료한 뒤 다시 실행해 DLL과 UI 자산이 함께 새로 로드되도록 합니다.
