# Tullius Widgets v1.2.1-rc.8

## 변경 요약
- 설정 패널에 PrismaUI 포커스가 들어온 직후, 런타임 visibility 판정이 그 상태를 일반 블로킹 UI로 오해해 커서를 다시 걷어가던 흐름을 수정했습니다.
- 이제 widget view가 실제로 focus를 잡고 있는 동안에는 pause/application/modal 계열 generic UI blocker 판정을 완화해, 설정 패널 커서가 즉시 사라지는 문제를 줄였습니다.
- `v1.2.1-rc.7`에서 보강한 hotkey task dispatch와 다음 틱 포커스 재시도는 그대로 유지됩니다.

## 사용자 영향/호환성
- 이번 RC는 `v1.2.1-rc.7` 이후 남아 있던 설정 패널 커서 소실 회귀를 바로잡는 추가 핫픽스입니다.
- 포커스 중인 widget view는 더 이상 자체 포커스 메뉴 때문에 HUD 숨김 판정에 휘말리지 않도록 조정됐습니다.
- 배포 구조와 설정 파일 형식은 유지됩니다. 네이티브 DLL과 Prisma UI 자산은 함께 업데이트하는 것을 권장합니다.

## 설치/업데이트 안내
1. `v1.2.1-rc.8` 프리릴리즈 zip을 다운로드합니다.
2. 압축 안의 `SKSE/`와 `PrismaUI/` 폴더를 게임의 `Data/` 폴더에 복사하거나 모드 매니저로 그대로 설치합니다.
3. 게임을 완전히 종료한 뒤 다시 실행해 DLL과 UI 자산이 함께 새로 로드되도록 합니다.
