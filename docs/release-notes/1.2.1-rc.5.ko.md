# Tullius Widgets v1.2.1-rc.5

## 변경 요약
- HUD 프런트 조립 구조를 정리해 `App.tsx`의 책임을 위젯 그룹/오버레이/표시 정책 단위로 분리했고, 경험치 위젯을 별도 컴포넌트로 추가해 표시 안정성을 높였습니다.
- 네이티브 런타임은 view bridge, 메뉴 숨김 상태, heartbeat/visibility 흐름을 모듈로 분리해 `main.cpp`와 런타임 오케스트레이션의 유지보수 부담을 줄였습니다.
- `StatsCollector`는 payload 정의, 수집 단계, JSON writer를 분리해 stats 계약 변경 시 영향 범위를 더 읽기 쉽게 정리했습니다.
- Windows 검증/패키징 흐름을 다듬어 WSL UNC worktree에서도 `verify-runtime-windows.ps1`와 prerelease packaging 경로를 더 일관되게 사용할 수 있게 했습니다.

## 사용자 영향/호환성
- 이번 RC는 기능 추가보다 구조 안정화와 릴리즈 검증 흐름 강화에 초점을 맞춘 빌드입니다.
- XP 위젯은 이전보다 넓은 표시 폭과 tooltip 보조 정보를 사용하지만, 매우 긴 값은 HUD 레이아웃 특성상 말줄임으로 보일 수 있습니다.
- 포토모드/캡처 계열 메뉴 숨김은 `photo`, `screenshot`, `capture`, `freecamera` 계열 휴리스틱과 UI blocking state를 함께 사용합니다.
- 배포 구조와 설정 파일 형식은 유지되며, 네이티브 DLL과 Prisma UI 자산은 함께 업데이트하는 것을 권장합니다.

## 설치/업데이트 안내
1. `v1.2.1-rc.5` 프리릴리즈 zip을 다운로드합니다.
2. 압축 안의 `SKSE/`와 `PrismaUI/` 폴더를 게임의 `Data/` 폴더에 복사하거나 모드 매니저로 그대로 설치합니다.
3. 게임을 완전히 종료한 뒤 다시 실행해 DLL과 UI 자산이 함께 새로 로드되도록 합니다.
