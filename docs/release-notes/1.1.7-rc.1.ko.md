# Tullius Widgets v1.1.7-rc.1

## 변경 요약
- 위젯 크기 옵션에 `매우 작음`을 추가해 1920x1080 환경에서 더 컴팩트하게 사용할 수 있도록 개선했습니다.
- 레벨업 직후 경험치(XP) 바 반영 지연 문제를 보강해, 메뉴 종료 후 지연 동기화로 안정적으로 최신 값을 반영합니다.
- UI 수치 표시에 음수 값이 노출되던 케이스를 보정했습니다.
- 플러그인 코어를 모듈화(`NativeStorage`, `RuntimeDiagnostics`, `WidgetEvents`, `WidgetHotkeys`, `WidgetJsListeners`, `WidgetBootstrap`)해 유지보수성과 안정성을 강화했습니다.

## 사용자 영향/호환성
- 기존 설정 파일(`Data/SKSE/Plugins/TulliusWidgets.json`) 및 프리셋(`TulliusWidgets_preset.json`)과 하위 호환됩니다.
- Prisma UI 브릿지(`updateStats`, `updateSettings`)의 기존 호출 규약을 유지합니다.
- 이번 버전은 프리릴리즈(`rc`)로, 실사용 전 기존 파일 백업을 권장합니다.

## 설치/업데이트 안내
1. `TulliusWidgets-v1.1.7-rc.1-ci.zip` 파일을 다운로드합니다.
2. 압축을 해제한 뒤 `Skyrim Special Edition/Data/` 경로에 파일을 덮어씁니다.
3. 업데이트 후 게임을 완전히 재시작해 DLL/UI가 함께 갱신되도록 해주세요.
