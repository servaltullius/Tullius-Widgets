# Tullius Widgets v1.1.7-rc.1

## 변경 요약
- 레벨업 직후 경험치(XP) 바 반영 지연 문제를 추가 보강했습니다.
  - 메뉴 종료 시점 이후 한 번 더 지연 동기화를 수행해 경험치/다음 레벨 필요치가 늦게 반영되는 케이스를 줄였습니다.
- 프리셋 가져오기/내보내기 안정성을 강화했습니다.
  - 과도한 크기의 프리셋 payload를 방어하고, 실패 시 결과 콜백을 일관되게 반환하도록 정리했습니다.
- 플러그인 코어를 모듈화(`NativeStorage`, `RuntimeDiagnostics`, `WidgetEvents`, `WidgetHotkeys`, `WidgetJsListeners`, `WidgetBootstrap`)해 유지보수성을 개선했습니다.
  - 내부 구조 정리 성격의 변경이며 기존 JS 브릿지 호출 규약은 유지됩니다.

## 사용자 영향/호환성
- 기존 설정 파일(`Data/SKSE/Plugins/TulliusWidgets.json`) 및 프리셋(`TulliusWidgets_preset.json`)과 하위 호환됩니다.
- Prisma UI 브릿지(`updateStats`, `updateSettings`)의 기존 호출 규약을 유지합니다.
- 이번 버전은 프리릴리즈(`rc`)로, 실사용 전 기존 파일 백업을 권장합니다.

## 설치/업데이트 안내
1. `TulliusWidgets-v1.1.7-rc.1-ci.zip` 파일을 다운로드합니다.
2. 압축을 해제한 뒤 `Skyrim Special Edition/Data/` 경로에 파일을 덮어씁니다.
3. 업데이트 후 게임을 완전히 재시작해 DLL/UI가 함께 갱신되도록 해주세요.
