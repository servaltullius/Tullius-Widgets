# Tullius Widgets v1.2.1-rc.4

## 변경 요약
- 전투 HUD 위젯의 기본 배치를 화면 높이와 위젯 크기까지 고려하도록 조정해, 짧은 해상도에서 아래 그룹이 잘리거나 겹치던 문제를 줄였습니다.
- 체력/매지카/스태미나와 소지 무게 위젯에 퍼센트 보조 정보와 상태 미터를 추가해, 위험 구간을 더 빨리 읽을 수 있도록 개선했습니다.
- 지속 효과 목록을 디버프 우선, 긴급 효과 강조, 진행 상태 표시 방식으로 다듬어 전투 중 가독성을 높였습니다.
- 저항 캡 표시, `F11` 전체 토글, 비동기 설정 저장, 런타임 계약 테스트/릴리즈 검증 흐름을 함께 보강했습니다.

## 사용자 영향/호환성
- 이번 RC는 위젯 가독성 개선과 릴리즈 안정화에 초점을 맞춘 빌드입니다.
- 네이티브 DLL과 Prisma UI 자산을 함께 업데이트하는 기존 배포 구조는 유지됩니다.
- 설정 파일 형식은 유지되며, 저장 실패 시 재시도/경고 동작이 이전보다 명확해졌습니다.

## 설치/업데이트 안내
1. `v1.2.1-rc.4` 배포 zip을 다운로드합니다.
2. 압축 안의 `SKSE/`와 `PrismaUI/` 폴더를 게임의 `Data/` 폴더에 복사하거나 모드 매니저로 그대로 설치합니다.
3. 게임을 완전히 종료한 뒤 다시 실행해 DLL과 UI 자산이 함께 새로 로드되도록 합니다.
