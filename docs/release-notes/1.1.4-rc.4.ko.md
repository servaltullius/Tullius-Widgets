# Tullius Widgets v1.1.4-rc.4

## 변경 요약
- 경험치(XP) 위젯 아이콘을 신규 리소스로 교체했습니다.
- 아이콘 교체에 맞춰 프론트엔드 번들 산출물을 갱신했습니다.

## 사용자 영향/호환성
- 기능 동작(경험치 계산/표시 로직) 변경은 없고, 아이콘 비주얼만 변경됩니다.
- 기존 저장 설정(`Data/SKSE/Plugins/TulliusWidgets.json`) 및 위젯 배치와 호환됩니다.

## 설치/업데이트 안내
- 기존 버전 위에 `TulliusWidgets-v1.1.4-rc.4.zip` 내용을 그대로 덮어쓰면 됩니다.
- 업데이트 후 게임 재시작(또는 모드 재로딩) 시 새 경험치 아이콘이 적용됩니다.
