# Tullius Widgets v1.1.3-rc.10

## 변경 요약
- 지속 효과(버프/디버프) 데이터 파이프라인을 안정화했습니다.
  - `effectName` 추출 시 주문 이름 fallback 복구
  - 식별자 없는 항목은 보수적 키로 분리해 오병합 방지
  - 부분/빈 stats payload 수신 시 이전 정상 상태를 유지하도록 정규화 강화
- C++ JSON 이스케이프를 강화해 제어문자 포함 이름에서도 파싱 실패 가능성을 줄였습니다.
- 화면 체력 경고 펄스가 설정 임계값(`lowHealthThreshold`)을 정확히 따르도록 수정했습니다.

## 사용자 영향/호환성
- 서로 다른 효과가 같은 이름으로 덮여 보이거나 중복처럼 보이는 현상이 완화됩니다.
- 로드 직후/전환 시점에 간헐적으로 위젯이 비정상 상태가 되는 가능성이 줄어듭니다.
- 기존 설정 파일 경로 및 포맷과 호환됩니다.

## 설치/업데이트 안내
1. `v1.1.3-rc.10` ZIP으로 기존 파일을 덮어씌워 업데이트합니다.
2. 게임에서 버프/디버프 목록(이름, 남은시간, 중복 행)을 확인합니다.
3. 동일 증상이 남아 있으면 해당 버프 조합과 스크린샷을 함께 알려주세요.
