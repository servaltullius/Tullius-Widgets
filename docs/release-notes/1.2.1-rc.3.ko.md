# Tullius Widgets v1.2.1-rc.3

## 변경 요약
- 레벨업 시 경험치(XP) 위젯 텍스트가 잘리던 문제를 수정했습니다.
- 텍스트 컨테이너의 flex 축소 동작을 조정해 WebKit 계열에서 수치가 눌리는 현상을 줄였습니다.
- XP 값 span의 표시 방식을 보강해 콘텐츠 크기 계산이 안정적으로 유지되도록 했습니다.

## 사용자 영향/호환성
- 이번 버전은 UI 표시 보정 중심의 RC 빌드입니다.
- 네이티브 DLL과 Prisma UI 자산을 함께 업데이트하는 기존 배포 구조는 유지됩니다.
- 기존 설정 파일 경로와 저장 형식은 바뀌지 않습니다.

## 설치/업데이트 안내
1. `v1.2.1-rc.3` 배포 zip을 다운로드합니다.
2. 압축 안의 `SKSE/`와 `PrismaUI/` 폴더를 게임의 `Data/` 폴더에 복사하거나 모드 매니저로 그대로 설치합니다.
3. 게임을 완전히 종료한 뒤 다시 실행해 DLL과 UI 자산이 함께 새로 로드되도록 합니다.
