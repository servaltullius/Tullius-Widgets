# Tullius Widgets v1.2.1-rc.9

## 변경 요약
- 설정 저장 동기화를 `revision` 기반 ACK로 정리해, 늦게 도착한 오래된 저장 성공/실패 응답이 최신 UI 상태를 덮어쓰지 않도록 보강했습니다.
- stats payload가 일시적으로 비거나 일부 필드가 비정상이어도 마지막 정상 HUD 상태를 유지하도록 복원력을 높였고, 의심스러운 브리지 계약은 한 번만 경고하도록 정리했습니다.
- 브릿지/핫키 문자열 상수화, 설정 패널 상태의 세션 저장, 공통 UI 토큰 정리로 네이티브와 프런트의 유지보수성을 함께 개선했습니다.

## 사용자 영향/호환성
- 이번 RC는 설정 저장 ACK 레이스와 HUD 데이터 일시 소실에 더 강한 빌드입니다.
- settings payload와 브리지 콜백은 하위 호환을 유지하면서 `revision` 정보를 추가로 다룹니다.
- 배포 구조는 유지되며, 네이티브 DLL과 Prisma UI 자산을 함께 업데이트하는 것을 권장합니다.

## 설치/업데이트 안내
1. `v1.2.1-rc.9` 프리릴리즈 zip을 다운로드합니다.
2. 압축 안의 `SKSE/`와 `PrismaUI/` 폴더를 게임의 `Data/` 폴더에 복사하거나 모드 매니저로 그대로 설치합니다.
3. 게임을 완전히 종료한 뒤 다시 실행해 DLL과 UI 자산이 함께 새로 로드되도록 합니다.
