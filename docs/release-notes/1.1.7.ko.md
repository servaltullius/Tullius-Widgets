# Tullius Widgets v1.1.7

## 변경 요약
- 레벨업 직후 경험치(XP) 바가 늦게 갱신되던 문제를 집중 개선했습니다.
  - 메뉴 전환/종료 직후 동기화 타이밍을 보강해, 레벨업 후 값 반영이 한 템포 늦는 체감을 줄였습니다.
- 전투 중 스탯 동기화 경로를 안정화하고 성능 부담을 낮췄습니다.
  - 업데이트 파이프라인을 `fast/full` 주기로 분리해, 빈번한 구간에서도 불필요한 전체 계산을 줄였습니다.
  - 지연 동기화 스케줄러를 단일 due-time 방식으로 정리해 반영 누락/지연 가능성을 낮췄습니다.
- 설정 저장과 프리셋 처리 안정성을 강화했습니다.
  - 설정 저장을 비동기(coalescing) 방식으로 바꿔 연속 설정 변경 시 프레임 드랍 가능성을 완화했습니다.
  - 프리셋 import/export 실패 처리와 payload 가드를 강화해 실패 시 피드백이 더 일관되게 동작합니다.
- UI 브릿지 호환성과 내부 유지보수성을 개선했습니다.
  - 기존 브릿지 키(`updateStats`, `updateSettings`)는 유지하면서 `window.TulliusWidgetsBridge.v1.*` 네임스페이스를 추가했습니다.
  - 내부 모듈 구조를 정리해 이후 기능 추가/버그 수정 시 변경 영향 범위를 줄였습니다.

## 사용자 영향/호환성
- 기존 설정 파일(`Data/SKSE/Plugins/TulliusWidgets.json`)과 프리셋(`TulliusWidgets_preset.json`)은 그대로 사용 가능합니다.
- 기존 Prisma UI 브릿지 호출 규약은 유지되며, 신규 네임스페이스는 확장 호환용입니다.
- 고빈도 fast 동기화 payload에서는 `timedEffects`가 생략될 수 있으며, UI는 이전 효과 목록을 유지하도록 동작합니다.
- 하위호환을 유지한 안정화/리팩터링 중심 릴리즈로, 사용자 설정 초기화나 수동 마이그레이션은 필요하지 않습니다.

## 설치/업데이트 안내
1. 릴리즈 페이지에서 `TulliusWidgets-v1.1.7-ci.zip` 파일을 다운로드합니다.
2. 압축을 해제한 뒤 `Skyrim Special Edition/Data/` 경로에 파일을 덮어씁니다.
3. 게임을 완전히 종료 후 재실행해 DLL/UI가 함께 갱신되도록 적용합니다.
4. 업데이트 후 레벨업 직후 XP 바 반영, 설정 저장, 프리셋 import/export 동작을 한 번 확인해 주세요.
