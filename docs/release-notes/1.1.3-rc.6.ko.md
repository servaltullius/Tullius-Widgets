## 변경 요약
- PrismaUI 뷰 생성 실패/무효 핸들 상황에서 위젯 초기화를 안전하게 중단하도록 런타임 가드를 강화했습니다.
- 설정 저장/프리셋 내보내기 시 `Data/SKSE/Plugins` 디렉터리를 자동 보장하고, 실패 시 로그/결과 콜백을 명확히 반환하도록 개선했습니다.
- 설정 import JSON 검증(타입/범위/섹션)을 추가해 손상된 설정 파일로 인한 설정 패널 오류를 방어했습니다.
- 지속 버프/디버프 리스트에 안정 키(`stableKey`)를 도입해 빈번한 갱신에서 재마운트/깜빡임을 줄였습니다.
- 드래그 이벤트 리스너와 프리셋 메시지 타이머 정리를 보강해 UI 안정성을 높였습니다.
- CI에 한국어 패치노트 필수 섹션 검증을 추가해 릴리즈 품질 게이트를 강화했습니다.

## 사용자 영향/호환성
- 대상: Skyrim SSE 1.5.97 / AE + SKSE 환경.
- 기존 브리지 키(`updateStats`, `updateSettings`, `importSettingsFromNative`)와 설정 경로(`Data/SKSE/Plugins/TulliusWidgets.json`)는 유지됩니다.
- 저장 데이터 포맷의 파괴적 변경은 없으며, 기존 사용자 설정은 안전 기본값과 병합되어 로드됩니다.

## 설치/업데이트 안내
1. 릴리즈 ZIP(`TulliusWidgets-v1.1.3-rc.6-ci.zip`)을 다운로드합니다.
2. zip 내용을 `Skyrim Special Edition/Data/` 폴더에 덮어씁니다.
3. 모드 매니저에서 구버전 `TulliusWidgets.dll` 중복 활성화를 해제한 뒤 게임을 실행합니다.
