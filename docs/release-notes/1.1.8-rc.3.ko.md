# Tullius Widgets v1.1.8-rc.3

## 변경 요약
- 위젯 수치 반영 정확도 개선 후속으로 **초기/전환 구간 오표시**와 **지연 반영**을 줄였습니다.
- **네이티브 갱신 주기 튜닝**
  - full 업데이트 주기를 단축해 빠른 전투/상태 변화에서 stale 구간을 줄였습니다.
  - heartbeat 주기와 pause 재시도 딜레이를 단축해 재동기화 반응성을 높였습니다.
  - 레벨업 메뉴 종료 후 강제 갱신 지연을 축소해 레벨업 직후 수치 반영 속도를 개선했습니다.
- **프론트 수치 표시 안전성 강화**
  - 라이브 `updateStats` 수신 전에는 위젯을 표시하지 않도록 변경했습니다.
  - 빈 payload(`{}`) 수신 시 라이브 상태를 내려 stale/목업 수치 노출 가능성을 줄였습니다.
  - 관련 훅 테스트를 추가해 빈 payload 처리 동작을 회귀 검증합니다.

## 사용자 영향/호환성
- 브릿지 계약(`updateStats`, `updateSettings`, `updateRuntimeStatus`)은 그대로 유지됩니다.
- 기존 설정/프리셋 파일 경로 및 포맷과 호환됩니다.
- 이번 RC는 기능 추가보다 **수치 표시 신뢰도와 반응성 개선** 중심 업데이트입니다.

## 설치/업데이트 안내
1. 릴리즈 페이지에서 `TulliusWidgets-v1.1.8-rc.3.zip` 파일을 다운로드합니다.
2. 압축을 해제한 뒤 `Skyrim Special Edition/Data/` 경로에 파일을 덮어씁니다.
3. 게임을 완전히 종료 후 재실행해 적용합니다.
