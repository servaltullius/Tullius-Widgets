# Tullius Widgets v1.1.3-rc.8

## 변경 요약
- 설정 패널을 탭/아코디언 구조로 개편해 긴 세로 스크롤 부담을 줄였습니다.
  - 탭: `일반`, `전투 수치`, `효과/시간`, `시각 알림`, `프리셋/배치`
  - 현재 탭 기준 `전체 펼치기/전체 접기` 지원
  - 마지막 탭/펼침 상태 기억
- 시간 위젯을 추가했습니다.
  - 스카이림 날짜/시간(4E 연도, 월/일, 시:분)
  - 현실 날짜/시간(초 단위 갱신)
- 스탯 갱신 안정성을 보강했습니다.
  - 메뉴 닫힘 직후 강제 갱신
  - heartbeat 주기 동기화(3초)

## 사용자 영향/호환성
- 기존 설정 파일(`Data/SKSE/Plugins/TulliusWidgets.json`)은 그대로 호환됩니다.
- 오래된 설정에는 새 항목(`time.gameDateTime`, `time.realDateTime`)이 기본값으로 자동 보완됩니다.
- 기존 위젯 배치/프리셋 흐름은 유지됩니다.

## 설치/업데이트 안내
1. 기존 버전 위에 `v1.1.3-rc.8` 파일을 덮어씁니다.
2. 게임 실행 후 `Insert`로 설정 패널을 열어 `효과/시간` 탭에서 시간 위젯 표시를 확인합니다.
3. 모드 매니저 사용 시 구버전 `TulliusWidgets.dll` 중복 활성화가 없는지 확인합니다.
