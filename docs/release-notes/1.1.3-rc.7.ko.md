## 변경 요약
- 지속 버프/디버프 위젯의 남은 시간(초) 표시를 이벤트 수신 간격에 의존하지 않도록 보강했습니다.
- 수신 시점 스냅샷(`snapshotAtMs`)을 기준으로 UI에서 로컬 카운트다운을 보간해, 초 단위 표시가 끊기거나 멈춰 보이는 현상을 완화했습니다.
- 남은 시간이 0초에 도달한 항목은 즉시 목록에서 정리되도록 표시 로직을 개선했습니다.

## 사용자 영향/호환성
- 대상: Skyrim SSE 1.5.97 / AE + SKSE 환경.
- 기존 JSON 브리지 키(`updateStats`, `updateSettings`)와 설정 파일 경로(`Data/SKSE/Plugins/TulliusWidgets.json`)는 그대로 유지됩니다.
- 저장 데이터/설정 포맷의 파괴적 변경은 없으며, 기존 설치에 덮어쓰기 업데이트가 가능합니다.

## 설치/업데이트 안내
1. 릴리즈 ZIP(`TulliusWidgets-v1.1.3-rc.7-ci.zip`)을 다운로드합니다.
2. zip 내용을 `Skyrim Special Edition/Data/` 폴더에 덮어씁니다.
3. 모드 매니저에서 구버전 `TulliusWidgets.dll` 중복 활성화를 해제한 뒤 게임을 실행합니다.
