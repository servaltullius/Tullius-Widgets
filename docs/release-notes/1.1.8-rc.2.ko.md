# Tullius Widgets v1.1.8-rc.2

## 변경 요약
- 심층 리뷰 후속 조치로 런타임 안정성, 임베디드 UI 호환성, 릴리즈 게이트를 추가 강화했습니다.
- **네이티브 안정성 개선**
  - Prisma UI DOM ready/JS listener 콜백을 게임 태스크 큐로 마샬링해 스레드 경계에서의 상태 접근 리스크를 줄였습니다.
  - `view` 핸들을 원자적으로 다루고 Interop 호출 전 유효성 확인 경로를 명확히 했습니다.
  - 통계 전송 경로의 pause 처리 정책을 통합해 이벤트/하트비트 간 동작 일관성을 높였습니다.
  - 설정/프리셋 파일 교체 시 롤백 가능한 교체 로직을 도입해 실패 상황의 파일 유실 가능성을 줄였습니다.
- **프론트엔드 호환성 개선**
  - 브릿지 초기화에서 최신 문법 의존(`??=`)을 제거해 임베디드 런타임 호환성을 높였습니다.
  - `structuredClone` 미지원 환경 대비 폴백을 추가했습니다.
  - 빌드 타깃을 하향 조정(`ES2020`/`es2019`)해 구형 엔진에서의 로드 실패 가능성을 완화했습니다.
- **CI/릴리즈 강화**
  - Windows 패키지 워크플로우에 `pull_request` 트리거를 추가했습니다.
  - 패키지/릴리즈 워크플로우에 frontend `lint + typecheck + test` 게이트를 포함했습니다.

## 사용자 영향/호환성
- 기존 설정 파일(`Data/SKSE/Plugins/TulliusWidgets.json`) 및 프리셋 파일과 호환됩니다.
- 브릿지 계약(`updateStats`, `updateSettings`, `updateRuntimeStatus`)은 유지되며, 외부 통합 인터페이스 변경은 없습니다.
- 이번 RC는 안정성/운영 품질 보강 목적의 업데이트입니다.

## 설치/업데이트 안내
1. 릴리즈 페이지에서 `TulliusWidgets-v1.1.8-rc.2.zip` 파일을 다운로드합니다.
2. 압축을 해제한 뒤 `Skyrim Special Edition/Data/` 경로에 파일을 덮어씁니다.
3. 게임을 완전히 종료 후 재실행해 적용합니다.
