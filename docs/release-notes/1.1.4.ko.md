# Tullius Widgets v1.1.4

## 변경 요약
- `v1.1.4-rc.1`부터 `v1.1.4-rc.10`까지의 안정화 변경을 정식 릴리즈에 반영했습니다.
- 경험치(XP) 위젯을 개선했습니다.
  - 단일 진행도 위젯(`현재 XP / 다음 레벨 총 필요 XP`)으로 통합하고, 단독 드래그 그룹으로 제공
  - `nextLevelTotalXp` 확장 필드 지원 + 구형 XP 토글 자동 마이그레이션 보강
  - 일부 상황에서 소수점 경험치 변화가 정수로 절삭되어 갱신이 누락될 수 있던 케이스를 보강
- UI/UX를 개선했습니다.
  - 크기 옵션에 `매우 작게`를 추가해 1920x1080 환경에서 더 작은 스케일로 조절 가능
  - 첫 실행 온보딩 카드 가독성 개선 및 `다시 보지 않기` 동작 정리
  - F11 임시 표시 상태와 설정의 표시 토글 UX 어긋남 완화
- 안정성/표시 정확도를 개선했습니다.
  - Prisma UI 브릿지 호출 타이밍을 DOM 준비 이후로 제한해 초기 로딩/세이브 로드 구간 안정화
  - 설정 저장을 임시 파일 작성 후 교체(원자적 갱신) 방식으로 변경 + 비정상적으로 큰 설정 파일 차단
  - 지속 효과(버프/디버프) 식별/중복 처리 보강
  - 장착 위젯 표시명 수집/왼손 판별 개선 + 장착 직후 1틱 지연 갱신으로 반영 일관성 향상

## 사용자 영향/호환성
- 기존 설정 파일(`Data/SKSE/Plugins/TulliusWidgets.json`) 및 프리셋(`TulliusWidgets_preset.json`)과 하위 호환됩니다.
- Prisma UI 브릿지(`updateStats`, `updateSettings`) 스키마는 하위 호환을 유지합니다.
  - `nextLevelTotalXp`는 확장 필드로 추가되었으며, 없는 환경에서도 정상 동작합니다.
- 참고: `매우 작게`로 설정한 뒤 이전 버전으로 되돌리면(다운그레이드) 크기 값이 기본값으로 되돌아갈 수 있습니다.

## 설치/업데이트 안내
1. `TulliusWidgets-v1.1.4.zip`을 다운로드합니다.
2. 압축을 해제한 뒤 `Skyrim Special Edition/Data/` 경로에 파일을 덮어씁니다.
3. 업데이트 후 게임을 완전히 재시작해 DLL/UI가 함께 갱신되도록 해주세요.
