## 변경 요약
- Skyrim SE 1.5.97에서 로드/초기 프레임 구간에 드물게 발생하던 CTD를 완화하는 핫픽스를 적용했습니다.
- 액티브 이펙트 조회 경로를 런타임 오프셋 안전 경로(`AsMagicTarget`) 기준으로 방어했습니다.
- 저장/로드 전환 중 불안정 프레임에서 지속효과 목록 수집을 건너뛰도록 보강했습니다.

## 사용자 영향/호환성
- 대상: Skyrim SSE 1.5.97 / SKSE 2.0.20 환경.
- 저장 파일 형식이나 설정 파일 경로(`Data/SKSE/Plugins/TulliusWidgets.json`) 변경은 없습니다.
- 기존 설치에서 DLL 교체만으로 업데이트 가능합니다.

## 설치/업데이트 안내
1. 릴리즈 ZIP(`TulliusWidgets-v1.1.3-rc.5-ci.zip`)을 다운로드합니다.
2. zip 내용을 `Skyrim Special Edition/Data/` 폴더에 덮어씁니다.
3. 모드 매니저에서 구버전 `TulliusWidgets.dll` 중복 활성화를 해제한 뒤 실행합니다.
