# Tullius Widgets v1.1.8-rc.6

## 변경 요약
- 장착 무기 위젯이 무기 교체 시 즉시 갱신되지 않던 문제를 수정했습니다.
  - 장비 변경 이벤트 후 200ms 후속 갱신을 추가해 게임 엔진의 장비 슬롯 반영 지연을 보완합니다.
  - 기존에는 최대 2초(하트비트) 대기 후에야 갱신되었습니다.
- 무기 교체 시 이전 무기 이름이 새 무기 이름과 겹쳐 보이던 렌더링 문제를 수정했습니다.
  - 무기 변경 시 React가 DOM 노드를 완전히 교체하여 WebKit의 이전 텍스트 잔상을 방지합니다.
  - 텍스트 오버플로 처리(`text-overflow: ellipsis`)가 임베디드 WebKit에서 정확히 동작하도록 CSS를 보강했습니다.
- 독/질병 저항력의 인위적인 캡 표시를 제거했습니다.
  - 기존에는 독 저항 85%, 질병 저항 100%로 클램프되어 표시되었으나, 이제 실제 액터 값을 그대로 표시합니다.
  - 캡 도달 시 금색 표시 및 경고 문구도 함께 제거했습니다.
- 체력/매지카/스태미나가 음수로 표시되던 문제를 수정했습니다.
  - 데미지 모디파이어가 기본값을 초과하는 경우에도 최소 0으로 클램프합니다.

## 사용자 영향/호환성
- C++ 네이티브 + 프론트엔드 양쪽 변경이므로 DLL과 프론트엔드를 함께 업데이트해야 합니다.
- 기존 설정 파일과 완전 호환됩니다.
- 위젯 레이아웃이나 기능 변경은 없습니다.

## 설치/업데이트 안내
1. 프리릴리즈 페이지에서 `v1.1.8-rc.6` zip을 다운로드합니다.
2. `Data/` 폴더에 덮어쓴 뒤 게임을 완전히 종료/재실행합니다.
3. 독/질병 저항이 85% 이상 올라가는지, 체력이 음수로 표시되지 않는지 확인합니다.
