# Tullius Widgets v1.1.3-rc.9

## 변경 요약
- 버프/디버프 위젯 정규화 로직을 보강했습니다.
  - 효과 이름 문자열의 제어문자 제거 및 공백 정리
  - 동일 효과가 중복 수신될 때 논리 키 기반 병합
  - React 렌더 키(`stableKey`) 충돌 완화로 중복 행/깜빡임 완화

## 사용자 영향/호환성
- 간헐적으로 발생하던 버프 텍스트 깨짐, 동일 버프 중복 표시 가능성이 줄어듭니다.
- 기존 저장 설정 파일과 호환됩니다.
- 표시 로직 보강이며, 게임 데이터 원본에는 영향을 주지 않습니다.

## 설치/업데이트 안내
1. `v1.1.3-rc.9` 파일로 덮어씌워 업데이트합니다.
2. 게임 실행 후 버프/디버프 위젯에서 텍스트와 중복 표시를 확인합니다.
3. 문제가 계속되면 스크린샷과 함께 재현 상황(버프 이름/발생 시점)을 알려주세요.
