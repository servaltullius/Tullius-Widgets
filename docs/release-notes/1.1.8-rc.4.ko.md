# Tullius Widgets v1.1.8-rc.4

## 변경 요약
- 네이티브-프론트 브리지 동기화 안정성을 강화했습니다.
  - stats payload에 `schemaVersion`/`seq` 메타를 추가해 역순 payload를 방어합니다.
  - settings payload에 `schemaVersion`/`rev` 처리를 추가해 tolerant parsing과 stale revision 무시를 보강했습니다.
  - settings 저장 결과를 `onSettingsSyncResult`로 UI에 전달해 저장 실패를 즉시 확인할 수 있습니다.
- 프론트 브리지 등록/해제 로직을 공통화(`window.*` + `TulliusWidgetsBridge.v1.*`)해 중복과 누수 위험을 줄였습니다.
- 런타임 검증 도구를 추가했습니다.
  - Windows 검증 스크립트: `scripts/verify-runtime-windows.ps1`
  - 검증 리포트 템플릿: `docs/plans/runtime-verification-report-template.ko.md`

## 사용자 영향/호환성
- 기존 브리지 키(`updateStats`, `updateSettings`)는 유지됩니다.
- 기존 설정 파일 경로/포맷과 호환되며, 신규 메타 필드는 구버전에서 무시 가능한 확장 방식입니다.
- settings 저장 실패 시 상단 경고 배너가 노출되어 문제를 즉시 인지할 수 있습니다.

## 설치/업데이트 안내
1. 프리릴리즈 페이지에서 버전 태그 `v1.1.8-rc.4`를 확인합니다.
2. Windows 빌드 환경에서는 `pwsh -File .\scripts\verify-runtime-windows.ps1 -CreateLocalPackage`로 패키지를 생성해 배포 경로에 반영합니다.
3. 기존 설치에 덮어쓴 뒤 게임을 완전히 종료/재실행해 적용합니다.
