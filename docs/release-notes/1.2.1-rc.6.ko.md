# Tullius Widgets v1.2.1-rc.6

## 변경 요약
- 설정 패널을 열 때 PrismaUI view 포커스 경로를 보강해, 게임 내에서 마우스 커서가 나타나지 않아 패널을 조작할 수 없던 문제를 수정했습니다.
- UI와 네이티브 사이에 설정 패널 열림/닫힘 상태를 동기화하는 브리지를 추가해, Insert/ESC hotkey 동작이 실제 패널 상태 기준으로 더 안정적으로 맞물리도록 정리했습니다.
- 포토모드/캡처 계열 메뉴 이름 휴리스틱에 더해 free camera 상태도 함께 감지하도록 바꿔, 포토모드 진입 시 위젯이 남아 있던 문제를 줄였습니다.

## 사용자 영향/호환성
- 이번 RC는 `v1.2.1-rc.5` 이후 게임 내 상호작용 회귀를 바로잡는 핫픽스 성격입니다.
- 설정 패널을 열면 이전보다 마우스 입력을 더 안정적으로 받을 수 있어야 하며, 포토모드/free camera 상태에서는 HUD가 숨겨지는 동작을 기대할 수 있습니다.
- 배포 구조와 설정 파일 형식은 유지됩니다. 네이티브 DLL과 Prisma UI 자산은 함께 업데이트하는 것을 권장합니다.

## 설치/업데이트 안내
1. `v1.2.1-rc.6` 프리릴리즈 zip을 다운로드합니다.
2. 압축 안의 `SKSE/`와 `PrismaUI/` 폴더를 게임의 `Data/` 폴더에 복사하거나 모드 매니저로 그대로 설치합니다.
3. 게임을 완전히 종료한 뒤 다시 실행해 DLL과 UI 자산이 함께 새로 로드되도록 합니다.
