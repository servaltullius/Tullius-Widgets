# Tullius Widgets v1.1.4-rc.5

## 변경 요약
- 지속 버프/디버프 식별 로직을 강화해 서로 다른 효과가 같은 이름으로 덮어 보이거나 중복 표시되는 문제를 완화했습니다.
- `값 변화 시에만 표시` 모드의 변화 감지 대상을 실제로 표시 중인 위젯 중심으로 조정했습니다.
- F11 임시 표시 상태와 설정 패널의 표시 토글 상태가 어긋나던 UX를 개선했습니다.
- 설정/프리셋 저장 경로를 게임 루트 기준 절대경로로 통일해 실행 경로 의존성을 줄였습니다.
- 경험치 브릿지 스키마에 `nextLevelTotalXp`를 추가하고, UI를 `현재 XP / 다음 레벨 총 필요 XP` 표기로 안정화했습니다.

## 사용자 영향/호환성
- 기존 설정 파일(`Data/SKSE/Plugins/TulliusWidgets.json`) 및 프리셋(`TulliusWidgets_preset.json`)과 하위 호환됩니다.
- 지속효과 리스트의 항목 식별/정렬이 더 안정적으로 동작하며, 특히 이름 중복/덮어쓰기 상황이 줄어듭니다.
- 기존 `experience`, `expToNextLevel` 필드는 유지되며, `nextLevelTotalXp`는 확장 필드로 추가됩니다.

## 설치/업데이트 안내
- 기존 버전 위에 `TulliusWidgets-v1.1.4-rc.5.zip` 내용을 그대로 덮어쓰면 됩니다.
- 업데이트 후 게임을 완전히 재시작해 새 DLL과 UI 번들이 함께 로드되도록 해주세요.
- 문제가 계속되면 구버전 `TulliusWidgets.dll` 중복 활성화 여부를 모드 매니저에서 확인하세요.
