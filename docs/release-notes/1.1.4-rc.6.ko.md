# Tullius Widgets v1.1.4-rc.6

## 변경 요약
- 첫 실행 온보딩 안내 팝업의 가독성을 높이기 위해 카드 크기, 여백, 제목/본문 폰트, 버튼 크기를 확대했습니다.
- 온보딩의 `다시 보지 않기` 동작을 개선해, 버튼을 누르지 않아도 설정 패널을 한 번 열면 자동으로 다시 표시되지 않도록 변경했습니다.
- `설정 열기` 버튼은 패널 오픈 역할에 집중하도록 단순화해, 온보딩 상태 처리를 패널 오픈 이벤트 기준으로 일관되게 정리했습니다.

## 사용자 영향/호환성
- 기존 설정 파일(`Data/SKSE/Plugins/TulliusWidgets.json`)과 완전 호환됩니다.
- 온보딩 팝업은 첫 노출 이후 설정 패널을 열기만 해도 자동으로 숨김 상태가 저장됩니다.
- 전투/스탯 계산, 버프/디버프 리스트, 경험치 위젯의 데이터 계산 로직에는 변경이 없습니다.

## 설치/업데이트 안내
- 기존 버전 위에 `TulliusWidgets-v1.1.4-rc.6.zip` 내용을 그대로 덮어쓰면 됩니다.
- 업데이트 후 게임을 완전히 재시작해 새 DLL/UI 번들이 함께 적용되도록 해주세요.
- 온보딩을 다시 보고 싶다면 설정 JSON의 `general.onboardingSeen` 값을 수동으로 `false`로 바꾸면 됩니다.
