# Tullius Widgets v1.1.3

## 변경 요약
- `v1.1.3-rc.5`부터 `v1.1.3-rc.10`까지의 안정화 변경을 정식 릴리즈에 반영했습니다.
- 지속 효과(버프/디버프) 위젯 안정성을 개선했습니다.
  - 남은 시간(초) 카운트다운 갱신 안정화
  - 효과 이름 정규화 및 중복/오병합 완화
  - 식별자 기반 안정 키 처리 강화
- 데이터 수집/파싱 경로를 보강했습니다.
  - 부분/비정상 payload 수신 시 이전 정상 상태 유지
  - C++ JSON escape 강화로 제어문자 포함 이름 처리 안정화
- 설정/UX를 개선했습니다.
  - 설정 패널 탭/아코디언 구조 개편
  - 스카이림 시간 + 현실 시간 위젯 추가
  - 화면 경고 효과 임계값 동작 보정

## 사용자 영향/호환성
- 기존 설정 파일 경로(`Data/SKSE/Plugins/TulliusWidgets.json`)와 호환됩니다.
- Prisma UI 브릿지 키(`updateStats`, `updateSettings`) 호환성을 유지합니다.
- 이번 릴리즈는 안정화 중심 업데이트이며, 기존 설치에 덮어쓰기 업데이트가 가능합니다.

## 설치/업데이트 안내
1. 릴리즈 ZIP(`TulliusWidgets-v1.1.3-ci.zip`)을 다운로드합니다.
2. 압축을 해제한 뒤 `Skyrim Special Edition/Data/` 경로에 파일을 덮어씁니다.
3. 모드 매니저 사용 시 구버전 `TulliusWidgets.dll` 중복 활성화가 없는지 확인한 후 실행합니다.
