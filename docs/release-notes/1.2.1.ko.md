# Tullius Widgets v1.2.1

## 변경 요약
- `v1.2.1` 정식판에서는 경험치 위젯 잘림, 설정 패널 커서/포커스 회귀, 포토 모드/free camera 숨김 문제, WSL UNC 기반 Windows 빌드/릴리즈 경로를 `rc.3`부터 `rc.10`까지 순차적으로 다듬은 내용을 하나로 묶었습니다.
- 전투 HUD는 시간 위젯, 지속 버프/디버프, 시각 경고, 드래그 배치, 값 변화 기반 표시 모드까지 포함한 현재 구성을 기준으로 가독성과 상호작용 안정성을 보강했습니다.
- UI 다국어 처리를 외부 JSON 기반으로 전환해 `PrismaUI/views/TulliusWidgets/i18n/manifest.json`과 언어별 JSON 파일만으로 새 언어를 추가할 수 있게 했습니다.
- 한국어 Skyrim 날짜/시간 표기는 숫자 월 형식으로 정리했고, 긴 날짜 문자열이 시간 위젯에서 잘리지 않도록 표시 폭도 함께 보정했습니다.
- 이번 정식판 마감 과정에서 HUD stats dispatch 경합, 치명타 계산 fallback, 플레이어와 무관한 전투 이벤트 과잉 갱신, 릴리즈 시 stale DLL 재사용 가능성을 추가로 정리했습니다.
- GitHub 릴리즈/로컬 패키징 흐름은 WSL UNC worktree와 일반 Windows 경로 모두에서 더 일관되게 동작하도록 정리했고, pre-commit/문서/넥서스 설명도 현재 배포 구조에 맞게 갱신했습니다.

## 사용자 영향/호환성
- 기본 제공 언어는 한국어/영어지만, 이제 별도 DLL 재빌드 없이 외부 JSON 언어팩으로 다른 언어를 추가할 수 있습니다.
- 설정 패널은 이전 RC들에서 보고되던 커서 미표시/즉시 포커스 해제 문제가 완화됐고, 포토 모드/free camera 상태에서는 HUD 숨김 동작이 더 안정적으로 적용되어야 합니다.
- 한국어 HUD에서는 게임 날짜가 `4E 201년 8월 21일 14:35` 같은 숫자 월 형식으로 표시됩니다. 영어 및 커스텀 언어팩은 각 언어 JSON의 날짜 포맷을 계속 사용할 수 있습니다.
- 치명타 확률, 저항, 피해 감소 등은 기존처럼 안전 범위와 실효값 기준으로 표시되며, 계산 실패 시 잘못된 `0%`로 떨어질 가능성을 줄였습니다.
- WSL에서 작업하는 개발 환경은 직접 `xmake`를 호출하기보다 공식 스크립트(`package.sh`, `release-local.ps1`, `verify-runtime-windows.ps1`)를 사용하는 경로가 권장됩니다.

## 설치/업데이트 안내
1. `v1.2.1` 정식 릴리즈 zip을 다운로드합니다.
2. 압축 안의 `SKSE/`와 `PrismaUI/` 폴더를 게임의 `Data/` 폴더에 복사하거나 모드 매니저로 그대로 설치합니다.
3. 기존 RC 버전을 사용 중이었다면 게임을 완전히 종료한 뒤 다시 실행해 DLL과 UI 자산이 함께 새로 로드되도록 합니다.
4. 새 언어를 추가하려면 `Data/PrismaUI/views/TulliusWidgets/i18n/` 아래에 언어 JSON 파일을 넣고 `manifest.json`의 `languages` 항목을 함께 갱신합니다.
