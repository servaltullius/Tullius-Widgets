# Tullius Widgets v1.1.4-rc.2

## 변경 요약
- 경험치 표시를 기존 2개 위젯(현재 경험치, 레벨업까지 경험치)에서 단일 진행도 위젯으로 통합했습니다.
  - 표시 형식: `현재 경험치 / 다음 레벨 필요 총 경험치`
- 경험치 위젯을 플레이어 정보 그룹에서 분리해 단독 드래그 그룹으로 추가했습니다.
- 경험치 위젯 아이콘을 도로롱 PNG 대신 경험치에 맞는 전용 벡터 아이콘으로 변경했습니다.
- 기존 저장 설정과의 호환을 위해 구형 XP 토글(`playerInfo.experience`, `playerInfo.expToNextLevel`)을 신규 토글(`experience.enabled`)로 자동 마이그레이션하도록 보강했습니다.

## 사용자 영향/호환성
- 기존 설정 파일(`Data/SKSE/Plugins/TulliusWidgets.json`)은 그대로 사용 가능합니다.
- 이전 버전에서 저장된 XP 표시 토글은 자동으로 새 경험치 위젯 설정으로 이전됩니다.
- 기존 브릿지 키(`updateStats`, `updateSettings`)와 다른 위젯 동작은 유지됩니다.

## 설치/업데이트 안내
1. `TulliusWidgets-v1.1.4-rc.2.zip`을 다운로드합니다.
2. 압축 해제 후 `Skyrim Special Edition/Data/` 경로에 덮어씁니다.
3. 게임 실행 후 설정 패널에서 `전투 수치 > 경험치` 섹션과 새 경험치 단독 위젯 표시를 확인합니다.
