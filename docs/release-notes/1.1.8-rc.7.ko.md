# Tullius Widgets v1.1.8-rc.7

## 변경 요약
- 레벨업 후 경험치 위젯이 즉시 갱신되지 않던 문제를 수정했습니다.
  - 메뉴 닫힘 후 뷰 복원 시 500ms 후속 갱신을 추가해 레벨업 메뉴에서 반영이 지연되는 XP 데이터를 보완합니다.
  - 만료된 예약 갱신이 새 갱신 예약을 차단하던 스케줄러 버그를 수정했습니다.

## 사용자 영향/호환성
- C++ 네이티브 변경이므로 DLL을 함께 업데이트해야 합니다.
- 기존 설정 파일과 완전 호환됩니다.
- 위젯 레이아웃이나 기능 변경은 없습니다.

## 설치/업데이트 안내
1. 프리릴리즈 페이지에서 `v1.1.8-rc.7` zip을 다운로드합니다.
2. `Data/` 폴더에 덮어쓴 뒤 게임을 완전히 종료/재실행합니다.
3. 레벨업 후 경험치 바가 즉시 갱신되는지 확인합니다.
