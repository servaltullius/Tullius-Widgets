# Tullius Widgets v1.1.4-rc.8

## 변경 요약
- 장착 위젯의 무기/장비 이름 수집 로직을 보강해, 인벤토리 표시명 기준으로 더 정확히 반영되도록 개선했습니다.
- 왼손 장비 판별 경로를 보완해 일부 환경에서 방패/장비가 누락되던 케이스를 줄였습니다.
- 장착 이벤트 처리 시 즉시 갱신 대신 한 틱 지연 갱신을 적용해, 장착 직후 이전 상태가 잠깐 보이는 문제를 완화했습니다.

## 사용자 영향/호환성
- 기존 설정 파일(`Data/SKSE/Plugins/TulliusWidgets.json`)과 호환됩니다.
- 위젯 구성/배치/토글 설정에는 변경이 없습니다.
- 장착 표시 안정성이 개선되어 무기/장비 전환 시 반영 일관성이 높아집니다.

## 설치/업데이트 안내
- 기존 버전 위에 `TulliusWidgets-v1.1.4-rc.8.zip` 내용을 그대로 덮어쓰면 됩니다.
- 업데이트 후 게임을 완전히 재시작해 DLL/UI가 함께 갱신되도록 해주세요.
- 장착 표시가 여전히 어긋나면 재현 상황(장비 종류/전환 순서)과 함께 로그를 제보해 주세요.
