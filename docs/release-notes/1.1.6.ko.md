# Tullius Widgets v1.1.6

## 변경 요약
- 레벨업 직후 경험치(XP) 위젯 반영이 늦어질 수 있던 케이스를 개선했습니다.
  - 레벨업 메뉴 종료 후 경험치/다음 레벨 필요 XP 값이 한 박자 늦게 갱신되는 경우를 대비해, 짧은 지연 후 스탯을 한 번 더 강제 동기화합니다.

## 사용자 영향/호환성
- 기존 설정 파일(`Data/SKSE/Plugins/TulliusWidgets.json`) 및 프리셋(`TulliusWidgets_preset.json`)과 하위 호환됩니다.
- Prisma UI 브릿지(`updateStats`, `updateSettings`) 스키마 변경은 없습니다.

## 설치/업데이트 안내
1. `TulliusWidgets-v1.1.6.zip`을 다운로드합니다.
2. 압축을 해제한 뒤 `Skyrim Special Edition/Data/` 경로에 파일을 덮어씁니다.
3. 업데이트 후 게임을 완전히 재시작해 DLL/UI가 함께 갱신되도록 해주세요.
