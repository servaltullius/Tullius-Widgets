# Tullius Widgets v1.1.4-rc.7

## 변경 요약
- Prisma UI 브릿지 호출 시점을 DOM 준비 이후로 엄격히 제한해 초기 로딩/세이브 로드 타이밍의 불안정 호출을 방지했습니다.
- 설정 파일 저장을 임시 파일 작성 후 교체하는 방식으로 바꾸고, 비정상적으로 큰 설정 파일을 차단해 설정 I/O 내구성을 강화했습니다.
- 지속 효과(버프/디버프) 식별 키에서 표시 텍스트 의존도를 낮춰, 서로 다른 효과가 같은 이름으로 덮여 보이거나 중복처럼 보이는 문제를 줄였습니다.
- JSON 문자열 이스케이프 로직을 공용 유틸(`src/JsonUtils.h`)로 통합해 데이터 직렬화 일관성을 높였습니다.

## 사용자 영향/호환성
- 기존 설정 파일(`Data/SKSE/Plugins/TulliusWidgets.json`) 경로와 포맷은 그대로 호환됩니다.
- 위젯 표시 구조는 유지되며, 버프/디버프 목록의 이름 꼬임/중복 표시 빈도가 감소합니다.
- 릴리즈 ZIP 루트 구조는 기존 정책대로 `SKSE/Plugins/...`, `PrismaUI/views/TulliusWidgets/...`를 유지합니다.

## 설치/업데이트 안내
- 기존 버전 위에 `TulliusWidgets-v1.1.4-rc.7.zip` 내용을 그대로 덮어쓰면 됩니다.
- 업데이트 후 게임을 완전히 재시작해 DLL/UI 번들이 함께 갱신되도록 해주세요.
- 문제가 재현되면 최신 `skse64.log`와 위젯 스크린샷(버프/디버프 영역 포함)을 함께 첨부해 제보해 주세요.
