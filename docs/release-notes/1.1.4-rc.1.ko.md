# Tullius Widgets v1.1.4-rc.1

## 변경 요약
- 플레이어 정보 위젯에 경험치 관련 항목 2개를 추가했습니다.
  - 현재 경험치(`experience`)
  - 레벨업까지 필요한 경험치(`expToNextLevel`)
- SKSE 수집 파이프라인에서 플레이어 스킬 런타임 데이터를 읽어 경험치 값을 전달하도록 확장했습니다.
- 설정 패널 `플레이어 정보` 섹션에 경험치 항목 토글 2개를 추가했습니다.
- 프론트 정규화/타입/목업 데이터를 업데이트해 기존 설정과 함께 안정적으로 동작하도록 보강했습니다.

## 사용자 영향/호환성
- 기존 설정 파일(`Data/SKSE/Plugins/TulliusWidgets.json`)과 호환됩니다.
- 새 옵션(`playerInfo.experience`, `playerInfo.expToNextLevel`)은 기본값으로 자동 보완됩니다.
- 기존 위젯 및 브릿지 키(`updateStats`, `updateSettings`) 호환성은 유지됩니다.

## 설치/업데이트 안내
1. `TulliusWidgets-v1.1.4-rc.1.zip`을 다운로드합니다.
2. 압축 해제 후 `Skyrim Special Edition/Data/` 경로에 덮어씁니다.
3. 게임 실행 후 설정 패널에서 `플레이어 정보 > 현재 경험치 / 레벨업까지 경험치` 표시를 확인합니다.
