# Tullius Widgets v1.1.4-rc.3

## 변경 요약
- 경험치 위젯 아이콘을 유저 제공 도로롱 경험치 아이콘으로 교체했습니다.
- 경험치 위젯은 기존처럼 `현재 경험치 / 다음 레벨 필요 총 경험치` 형식을 유지합니다.
- 로컬 패키징 스크립트에서 `dist/` 내부 기존 zip이 릴리즈 zip에 중첩 포함되지 않도록 정리 단계를 보강했습니다.

## 사용자 영향/호환성
- 설정/저장 파일 호환성에는 변경이 없습니다.
- UI 아이콘만 교체되며, 경험치 계산 및 표시 로직은 기존 `v1.1.4-rc.2`와 동일합니다.
- 기존 브릿지 키(`updateStats`, `updateSettings`) 호환성은 유지됩니다.

## 설치/업데이트 안내
1. `TulliusWidgets-v1.1.4-rc.3.zip`을 다운로드합니다.
2. 압축 해제 후 `Skyrim Special Edition/Data/` 경로에 덮어씁니다.
3. 게임 실행 후 경험치 단독 위젯 아이콘이 새 도로롱 경험치 아이콘으로 표시되는지 확인합니다.
